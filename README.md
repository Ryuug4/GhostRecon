## How to Use



1. **Clone the Repository:**

    `git clone https://github.com/Ryuug4/GhostRecon.git`

2. **Run the Script:**

    `sudo python3 ghostrecon.py -H <targetip/domain> -t All`

    `sudo python3 ghostrecon.py -H <targetip/domain> -t Recon`

   -H - Host/Target
   -t - task --->  Recon , All , UDP Scan and so on
